mod2addon is a tool designed to streamline the process of porting Minecraft Java Edition mods to Bedrock Edition addons. It focuses on generating addon templates and converting data-based assets.

Planned and supported features:

  • Texture and asset structure conversion
  • Recipe format conversion
  • Behavior Pack / Resource Pack template generation
  • Automatic reports for unsupported elements

Java code (Forge / Fabric APIs) cannot be converted automatically. This project aims to reduce manual work, not to provide full automation.
