mod2addon

Java Edition MOD → Bedrock Edition アドオン

mod2addon は、Minecraft Java Edition の MOD を Bedrock Edition のアドオンへ移植する作業を 効率化するためのツールです。 テンプレート生成やデータアセットの変換に対応します。

  • テクスチャ / アセット構造の変換
  • レシピ形式変換
  • Behavior / Resource Pack の雛形生成
  • 非対応要素の自動レポート

Javaコード（Forge / Fabric API）は自動変換できません。 本ツールは完全自動化ではなく、手作業の削減を目的としています。
